# NexSpend CLI

A CLI tool to analyze bank CSV files and generate simple financial summaries.


##  Motivation

Managing personal finances through raw bank statements can be confusing and inefficient.  
NexSpend CLI was created to transform raw CSV data into clear and useful financial insights, helping users better understand their spending habits.


##  Features

- Read bank CSV files  
- Process financial transactions  
- Calculate total expenses  
- Display a simple financial summary in the terminal  


##  Usage

1. Export your bank statement as a CSV file  
2. Run the script  
3. View your financial summary directly in the terminal
